# Meu Portfólio

Projeto pessoal de portfólio criado em HTML, CSS e JavaScript. A ideia é apresentar um layout moderno com navegação fixa, seção inicial animada e um cartão flutuante com destaque visual.

## O que tem aqui

- Header fixo com logo e menu de navegação
- Seção inicial (`#home`) com fundo em gradiente animado
- Caixa flutuante (`.floating-box`) com efeito de movimento
- Layout responsivo para telas menores

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript (estrutura de suporte)

## Como usar

1. Abra o arquivo `index.html` no navegador; ou
2. Execute um servidor local (por exemplo, Live Server no VS Code);
3. Visualize o layout e as animações na página inicial.

## Personalização

Você pode alterar cores, textos e animações diretamente em `css/styles.css` para ajustar o estilo do portfólio ao seu gosto.
