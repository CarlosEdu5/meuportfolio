# 💻 Personal Portfolio

Um portfólio desenvolvido para apresentar minha identidade como desenvolvedor, destacando projetos, habilidades e tecnologias por meio de uma interface moderna, responsiva e com foco na experiência do usuário.

## Sobre

Este projeto foi desenvolvido com o objetivo de criar uma presença digital que represente minha forma de desenvolver software: código limpo, atenção aos detalhes e interfaces bem construídas.

Cada seção foi pensada para transmitir não apenas os projetos que desenvolvi, mas também minha evolução técnica e meu interesse por criar experiências intuitivas e de alta qualidade.

## Tecnologias

* HTML5
* CSS3
* JavaScript (ES6+)

## Principais características

* Interface moderna e minimalista
* Layout totalmente responsivo
* Animações e transições suaves
* Código organizado e de fácil manutenção
* Estrutura otimizada para desempenho
* Compatibilidade com os principais navegadores

## Objetivo

Mais do que um site de apresentação, este portfólio funciona como um espaço para compartilhar projetos, demonstrar minhas habilidades em desenvolvimento front-end e acompanhar minha evolução como desenvolvedor.
